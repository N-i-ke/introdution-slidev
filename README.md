# introdution-slidev

このリポジトリは[Slidev](https://sli.dev/)を使ったプレゼンテーション資料の管理用です。

## セットアップ

1. Node.js（v14以上）をインストールしてください。
2. 以下のコマンドで依存関係をインストールします。

```sh
npm install
```

## スライドの起動

```sh
npx slidev
```

## スライドの編集

- `slides.md` をMarkdown形式で編集してください。
- Slidevの詳細は[公式ドキュメント](https://sli.dev/)を参照してください。
